# Install
```python
git clone https://github.com/kiahamedi/URL-Shortener.git
cd URL-Shortener/
pip install -r requirements.txt
cd app/
python manage.py migrate
```
<br>


# Run localhost
```python
python manage.py runserver
```
<br>
